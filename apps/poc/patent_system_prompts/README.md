# 特許データ保存システム 実装用プロンプト集

このZIPは、特許番号から請求項1（全文）を取得できる自前システムを、段階的に実装するためのプロンプト集です。

## 使い方（推奨）
1. まず `prompts/00_master_prompt.md` を、あなたの開発用AI（コード生成アシスタント）へ渡してください。
2. AIがリポジトリの骨組みを生成したら、次に `prompts/01_...` 以降を順番に渡して、各モジュールを埋めていってください。
3. 途中で仕様を変えるときは、該当プロンプトを修正して再投入してください。

## 想定スタック
- Python 3.12
- FastAPI（HTTP API）
- PostgreSQL（永続化）
- SQLAlchemy + Alembic（DBアクセスとマイグレーション）
- Typer（CLI）

## 対象範囲
- 日本特許（公報XML）を主対象
- バルク入手（またはローカル配置）→ Raw保管 → XMLパース → 請求項抽出 → API提供

## 重要な注意
- 公的サイトの画面スクレイピングやロボットアクセスは禁止されている場合があります。運用は、契約・規約・法令に従ってください。
- 本ZIPは実装を支援するための指示文です。データ入手の権限や提供条件は、利用者側で確認してください。
