# Welcome to My Documentation

Welcome to the documentation site! This is a template for a Docusaurus site hosted on AWS CloudFront.

## Getting Started

This documentation site is built with [Docusaurus](https://docusaurus.io/), a modern static website generator.

### Features

- **Fast**: Built on React, optimized for speed
- **Markdown**: Write docs in Markdown
- **Customizable**: Extensive theming options
- **Search**: Built-in search functionality
- **Versioning**: Document versioning support
- **i18n**: Internationalization support

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## What's Next?

Start adding your documentation pages in the `docs/` directory!
